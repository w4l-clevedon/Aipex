# Express Divorce Industry Pack

The Express Divorce pack is the first vertical implementation of Aipex Client OS.

## Purpose
Provide a client-facing legal case portal with staff/admin workflow controls.

## Reusable patterns
- Applicant/respondent/client roles
- Magic-link external access
- Case stages
- Missing-information requests
- Secure uploads
- Messages/tickets
- Approvals
- PDF/document generation
- Audit trail
- Admin Kanban
- Email automation

## Core workflow mapping
Express Divorce-specific case stages should be implemented as workflow templates, not hard-coded plugin logic.

## MVP priority
Use this pack to prove the generic workflow engine and client portal before extracting additional industry packs.
